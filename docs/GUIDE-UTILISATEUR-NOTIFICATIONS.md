# Guide Utilisateur — Notifications AssoAI v3.1

## 🔔 Centre de Notifications

La cloche en haut à droite affiche un badge rouge avec le nombre d'alertes non lues.

### Types de notifications

| Icône | Type | Signification |
|-------|------|---------------|
| 🔴 | Alerte | Retard critique, action requise |
| ⏰ | Rappel | Échéance approche, tâche à faire |
| 🚨 | Escalade | Retard persistant, escaladé au superviseur/Admin |
| ℹ️ | Info | Rapport quotidien, changement de phase |

### Actions

- **Cliquer** sur une notification → ouvre le projet concerné
- **Tout marquer lu** → badge remis à zéro
- Les notifications lues passent en gris

## 📊 Rapport Quotidien (8h)

Chaque matin à 8h, un rapport est généré automatiquement :
- Projets en retard 🔴
- Projets à surveiller 🟡
- Projets dans les temps 🟢

## 🚨 Escalade Progressive

| Délai | Action |
|-------|--------|
| < 24h | Silence (délai de grâce) |
| 24-48h | Notification à l'assigné |
| 48-72h | Notification + copie Superviseur |
| > 72h | Alerte critique à l'Admin |

## 📱 Accès Mobile

L'application est responsive. Tout est accessible depuis un navigateur mobile.
